# nishant-builds — personal portfolio

Built with [Hugo](https://gohugo.io/) + [PaperMod](https://github.com/adityatelange/hugo-PaperMod) theme.

Live at: https://nishant-builds.github.io

## Adding content

**New log entry:**
```
hugo new log/YYYY-MM-DD-title.md
```

**New learning:**
```
hugo new learnings/title.md
```

**New deep dive:**
```
hugo new deep-dives/title.md
```

**New project:**
```
hugo new projects/title.md
```

## Local preview
```
hugo server -D
```
Then open http://localhost:1313
